Αλλάξτε την εμφάνιση του Firefox με CSS

11 Nov 2008 by lexx

Όλοι γνωρίζετε ότι μπορείτε να εγκαθιστάτε add-ons και themes στον Firefox browser σας. Προσωπικά έχω βρει δεκάδες add-ons τα οποία έχουν κάνει τη ζωή μου πολύ ευκολότερη. Προσφάτως ανακάλυψα και το Stylish το οποίο δίνει πολλές νέες δυνατότητες στην παραμετροποίηση της εμφάνισης και του UI του Firefox γράφοντας μονάχα CSS!

To Stylish στην πραγματικότητα δεν είναι ένα ακόμα add-on. Είναι ένα νέο παράθυρο στο customization του Firefox. Όσοι ξέρετε CSS θα μπορείτε να του αλλάξετε τα φώτα. Όμως ακόμα και όσοι δεν γνωρίζουν μπορούν στο UserStyles.org να βρείτε έτοιμα styles που εγκαθιστώνται μόνο με ένα click.

Αυτό που μου άρεσε πολύ είναι ότι για να δείτε τις αλλαγές δεν χρειάζεται να επανεκκινήσετε των browser. Γίνονται όλα άμεσα και γρήγορα, και επειδή μιλάμε για CSS, τα styles δεν επιβαρύνουν καθόλου το σύστημα.

Πέρα από αλλαγές στο Firefox, τα Styles αλλάζουν και δημοφιλείς σελίδες και υπηρεσίες όπως το Gmail κλπ. Εδώ θα βρείτε και μαι λίστα με Applications που μπορείτε να αντικαταστήσετε με το πολύ ελαφρύτερο Stylish.
