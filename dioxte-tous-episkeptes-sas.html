Διώξτε τους επισκέπτες σας

22 Nov 2006 by lexx

Είναι πολύ απλό και εύκολο να σπάσετε τα νεύρα αυτών που επισκέπτονται το website σας. Αρκεί να ακολουθήσετε τα παρακάτω βήματα και θα είστε σίγουροι μετά ότι το site σας θα είναι μισητό :)

  1. Τα Pop-Up είναι ένας εξαιρετικός τρόπος για να ενοχλήσετε αρχικά τον επισκέπτη. Αρκεί να ανοίξει ένα για να χάσει το site σας την αξιοπιστία του και την σοβαρότητα του. Στην Ελλάδα έχει γίνει και μόδα σε μεγάλα site. Αυτό που θα σκεφτεί ο επισκέπτης: "Μα καλά πόσα τους πληρώνουν για να μου σπάνε τα νεύρα;"

  2. Οι σελίδες Υπό Κατασκευή είναι άλλο ένας τρόπος να κάνεις τον επισκέπτη να κάνει 2 παραπάνω ανούσια κλικ και να φανεί η σελίδα ερασιτεχνική. Αν είναι υπό κατασκευή καλύτερα να το γνωρίζει μόνο ο webmaster.

  3. Οι Ψεύτικες Δηλώσεις βοηθάνε επίσης στην αρνητική εικόνα. Ένας πολύ καλός τρόπος να ξεφτιλίσετε τη σελίδα σας είναι να λέτε πχ "Δωρεάν Themes" και μετά στη σελίδα να λέτε "Με 30 euro το χρόνο θα έχετε όσα δωρεάν theme θέλετε". Καλό;

  4. Με Ατέλειωτες Σελίδες σε μήκος ο χρήστης θα πρέπει να κάνει πολύ scroll. Ειδικά αν έχει τενοντίτιδα θα τον κάνετε να εξοργιστεί. Σας το συνιστώ ανεπιφύλαχτα.

  5. Βάζοντας να παίζει Αυτόματα Μουσική στον επισκέπτη της σελίδας σας είναι ένας από τους πιο σίγουρους και επιτυχημένους τρόπους να χαλάσετε εντελώς το κλίμα του. Επί τόπου χαλάει το playlist που ακούει ήρεμος κα μπλέκονται τα δύο τραγούδια δημιουργώντας ένα κομφούζιο. Εξαιρετικό!

  6. Υπερβολικά Γραφικά και πολλά banner. Τα συνιστώ για να αποκλείσετε όλους τους χρήστες με κακές συνδέσεις. Θα κάνει 10 λεπτά να ανοίξει η σελίδα και σίγουρα θα προτιμήσουν να πάνε κάπου αλλού. Δοκιμάστε το.

  7. Βάλτε Plug-Ins που πρέπει να κατεβάσει ο χρήστης για να μπορεί να δει τη σελίδα. Σίγουρα θα βαρεθεί και θα φύγει. 90% επιτυχία.

  8. Προσπαθήστε να μην υπάρχει καμία ομοιογένεια στις σελίδες του site σας έτσι ώστε ο χρήστης να μην νιώθει οικεία. Βάλτε πολλά διαφορετικά χρώματα και προσπαθήστε ο σχεδιασμός να θυμίζει κάποιο άλλο site από σελίδα σε σελίδα.

Αν ακολουθείτε μερικούς από αυτούς του κανόνες είστε σε καλό δρόμο. Αν συνεχίσετε, σίγουρα με την πάροδο του χρόνου θα διώξετε και του χρήστες που έχουν απομείνει.
