Html, Τext Editors και συγγραφή σε blog

09 Oct 2007 by lexx

Με αφορμή το άρθρο που διάβασα στο Something Digital θα γράψω μερικά πράγματα για τους διάφορους text editors που κυκλοφορούν. Θα πω ότι το άρθρο που διάβασα κάνει μια πολύ ωραία και χρήσιμη παρουσίαση για text editors όπου σας προτείνω να διαβάσετε αν σας ενδιαφέρει το θέμα.

Η δική μου εμπειρία από τους text editors μόνο καλή δεν είναι. Κάθε φορά που έχω χρησιμοποιήσει έναν μετά πηγαίνοντας να δω το html που έχουν δημιουργήσει αυτόματα θέλω να βάλω τα κλάματα. Πολλά περιττά tags ανακατεμένα. Πολλές φορές διπλά ή τριπλά κλπ.

Προσωπικά έχω σταματήσει στα blog μου να χρησιμοποιώ οποιονδήποτε editor εδώ και ένα χρόνο και κάτι. Προτιμώ να τα κάνω όλα με html και css. Πρώτον γιατί έχω πλήρη έλεγχο του περιεχομένου, καθαρό κώδικα χωρίς λάθη και όταν αποφασίσω να αλλάξω theme ξέρω ακριβώς τι global αλλαγές θα κάνω για να μείνει το στυλ των κειμένων.

Το σίγουρο είναι ότι κάποιος θα σκεφτεί ότι "Αν δεν ξέρω html τι θα κάνω;". Η απάντηση μου είναι "Επιτρέπεται να έχει blog και να μην έχεις βασικές html γνώσεις για την μορφοποίηση ενός κειμένου;". Απάντηση με ερώτηση μεν αλλά η html είναι παιχνιδάκι και υπόθεση ωρών για να την μάθει κάποιος τουλάχιστον για να μορφοποιεί το κείμενο του. Οπότε είναι καθαρά θέμα βαρεμάρας και απροθυμίας η μη εκμάθηση της. Και αυτά τα δύο χαρακτηριστικά δεν είναι χαρακτηριστικά ενός blogger.

Τα παραπάνω δεν τα λέω με κακία σε καμία περίπτωση και μπορεί όντως να υπάρχουν περιπτώσεις όπου ο text editor όντως να είναι χρήσιμος αλλά και πάλι θα πρέπει να ξέρετε html για να πάρετε το μορφοποιημένο κείμενο και να το καθαρίσετε. Σε ένα άλλο post θα εξηγήσω γιατί αξίζει να είναι καθαρός ο κώδικας σας.

Αν σας έπεισα ορίστε μια καλή αρχή για να μπείτε στην html.
