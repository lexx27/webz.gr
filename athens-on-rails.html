Athens On Rails

20 Oct 2008 by PanosJee

Η πρώτη συνάντηση αλλά και παρουσίαση του web framework Ruby On Rails είναι γεγονός! Την Παρασκευή 31 Οκτωβρίου στις 17:00 στο Οικονομικό Πανεπιστήμιο Αθηνών επαγγελματίες αλλά και fan boys του δημοφιλούς Open Source framework θα παρουσιάσουν στους Έλληνες developers τις δυνατότητες και τις χάρες του.

Σύντομο πρόγραμμα:

  • 17:00 Προσέλευση κοινού
  • 17:05 Εισαγωγή στη Ruby και το Rails
  • 17:30 O Παναγιώτης Παπαδόπουλος παρουσιάζει το κλασσικό demo, Creating a weblog in 20 minutes + extras
  • 18:00 Ο Νίκος Δημητρακόπουλος θα μιλήσει για τις βέλτιστες πρακτικές που εφαρμόζονται στο Rails όπως MVC, REST κλπ
  • 18:20 O Νικόλας Κόκκαλης, δημιουργός μερικών από των πιο διαδεδομένων Facebook Applications και ιδρυτής της Grebooca θα μιλήσει για Facebook development με Rails
  • 18:40 Ερωτήσεις και συζήτηση

Συμμετοχές και πληροφορίες στο αντίστοιχο Facebook event Ελπίζουμε να σας δούμε όλους εκεί!
