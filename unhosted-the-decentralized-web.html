Unhosted & the decentralized web

11 Dec 2011 by lexx

Το κίνημα της «αποκέντρωσης» των δεδομένων (το λέω καλά;) ή αλλιώς decentralized web είναι κάτι που μου έχει τραβήξει πολύ το ενδιαφέρον τελευταία. Δεν ξέρω αν το νιώθετε και εσείς αλλά έχω αρχίσει να νιώθω άβολα όταν το μεγαλύτερο ποσοστό των δεδομένων μου βρίσκονται στο google ή το facebook. Και αυτό που με κάνει να ανησυχώ περισσότερο είναι ότι η τάση αυτή, της συσσώρευσης των δεδομένων σε λίγες εταιρίες θηρία, αυξάνεται.

Τα καλά νέα βέβαια είναι ότι πάρα πολλοί άλλοι άνθρωποι έχουν παρόμοιες ανησυχίες. Έτσι έχει ξεκινήσει άλλη μια τάση. Αυτή του decentralized web, που έχει σαν σκοπό να βρει μια λύση ώστε να έχουμε περισσότερο έλεγχο στα προσωπικά μας δεδομένα.

Ένα project που έχει βγει από αυτή την ιστορία είναι το unhosted. Βασική φιλοσοφία είναι να σπάσει το μοντέλο «you get our app, we get your data». Δηλαδή, μιλάμε για τη δημιουργία ενός νέου μοντέλου όπου η εφαρμογή θα είναι στο web βασισμένη σε html και javascript αλλά δεν θα είναι «δεμένη» με κάποιον host. Τα δεδομένα θα αποθηκεύονται στο σημείο επιλογής (remote storage) του χρήστη. Φανταστείτε τα google docs για παράδειγμα να αποθηκεύονταν στον server σας αντί για τον server της google αλλά να μπορείτε να έχετε όλες τις φανταστικές δυνατότητες που προσφέρουν.

Για την ώρα το project είναι στα πρώτα του βήματα και δεν υπάρχει κάποια εφαρμογή της κλάσης του google docs. Το πιο κοντινό παράδειγμα στην φιλοσοφία του unhosted είναι το diaspora. Μπορείτε επίσης να δείτε διάφορα πειράματα που έχουν γίνει στο github.
