PHP Speedy - aggregate & compress Wordpress

15 Apr 2008 by lexx

Εγκατέστησα και τώρα γράφω για ένα plugin το οποίο θα κάνει πολύ πιο γρήγορο το blog σας.

Μια από τις εκκρεμότητες του v4 ήταν να περιορίσουμε λίγο τα loading times του WEBZ μιας και έχει αρχίσει να έχει αρκετά javascript και CSS. Την λύση βρήκα σε ένα φρέσκο και λίγο κρυμμένο plugin(γιατί δεν είναι στο wordpress.org).

Πριν πάρετε κάποια απόφαση θα σας πρότεινα να διαβάσετε το άρθρο που είχα γράψει: "Κάνε το Wordpress Blog σου να φορτώνει πιο γρήγορα".

PHP Speedy

Το PHP Speedy αρχικά ξεκίνησε σαν ένα script το οποίο κάνει aggregate και compress τα html, js και css αρχεία μιας σελίδας. Μην τρομάζετε, θα εξηγήσω.

Όταν λέω aggregate εννοώ ότι αν έχετε 5 javascript αρχεία, το πρόγραμμα θα τα συμμαζέψει όλα σε ένα αρχείο. Το αποτέλεσμα θα είναι να μειωθούν τα HTTP Requests. Ένας από τους βασικούς τρόπους για να γίνει μια σελίδα πιο "ελαφριά".

Όταν λέω compress εννοώ ότι το πρόγραμμα "τρώει" όλα τα κενά μέσα στο κάθε αρχείο με αποτέλεσμα να μικραίνει το filesize τους.

Ξεκίνησα βιαστικά να το εγκαθιστώ και ευτυχώς μέχρι να ανέβουν τα αρχεία στο ftp είδα ότι τον Μάρτιο που πέρασε είχε δημιουργηθεί ειδικό plugin για το Wordpress. Συμβατό μάλιστα με το 2.5. Τρέχει ήδη στο WEBZ.

H εγκατάσταση είναι η κλασική όπως σε κάθε Wordpress Plugin. Οι οδηγίες στην παραμετροποίηση είναι αναλυτικές και λογικά δεν θα αντιμετωπίσετε κάποιο πρόβλημα. Αφήνω μόνο ένα ερωτηματικό για το πόσο αυτή η διαδικασία επιβαρύνει των server. Θα το μάθω σύντομα και θα σας ενημερώσω.

15.4.08 edit:

Αντιμετώπισα πρόβλημα με τον Safari ο οποίος δεν έβρισκε το style. Κάτι με τα meta θα είναι λογικά το θέμα.

Επίσης το wp-polls δεν δούλευε. Κάποιο πρόβλημα λογικά με το javascript του ajax.
