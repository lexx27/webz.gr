URL URI IRI - Όλη η αλήθεια

08 Dec 2007 by lexx

Ήθελα σήμερα να γράψω ένα άρθρο για την σωστή χρήση των URI ως προς το SEO. Όμως στην πορεία της έρευνας μου έπεσα πάνω σε πολλές πληροφορίες οι οποίες μου ήταν άγνωστες και τις οποίες θέλω να μοιραστώ (είμαι πολύ καλός το ξέρω). Ποιες οι διαφορές μεταξύ URL, URI και IRI; Προσωπικά δεν τα είχα ξεκαθαρίσει στο μυαλό μου.

URL URN και URI

Γενικά στο διαδίκτυο συναντάμε συχνά τους όρους URL (Uniform Resource Locator) και URI (Uniform Resource Identifier), οι οποίοι συνήθως χρησιμοποιούνται από τους χρήστες για να προσδιορίσουν το ίδιο πράγμα, δηλαδή μια διαδικτυακή διεύθυνση (web address).

To W3C δεν αποδέχεται πλέον τον όρο URL και έχει διαλέξει σαν επίσημο τον όρο URI. Τεχνικά όπως φαίνεται και στο παράδειγμα ένα URL είναι πάντα URI, ενώ ένα URI δεν είναι πάντα URL.

Oι Διαφορές με παραδείγματα URL: http://www.webz.gr/forum/post.html URN: www.webz.gr/forum/post.html#comment URI: http://www.webz.gr/forum/post.html#comment

Οπότε από σήμερα έμαθα να μην χρησιμοποιώ των όρο URL όπως έκανα τόσα χρόνια. Κάτι είναι και αυτό. Πάμε λοιπόν να μιλήσουμε για τα IRI.

IRI (Internationalized Resource Identifier)

Το IRI στην ουσία είναι η διεθνοποιημένη "έκδοση" του URI. Επιτρέπει την χρήση και άλλων χαρακτήρων πέρα από US-ASCII, όπως Universal Character Set (Unicode).

Παρακάτω αυτοσχεδιάζω και ζητώ διόρθωση. Αν έχω καταλάβει λοιπόν καλά IRI είναι το παρακάτω για παράδειγμα:

http://www.webz.gr/2007/12/06/open-coffee-vi-δύο-είδη-κίνησης/

Αν ισχύει αυτό τότε ο Firefox 2 δεν υποστηρίζει ακόμα ολοκληρωμένα τα IRI σε αντίθεση με τον Internet Explorer 7. OMG, ποτέ δεν περίμενα να πω καλό λόγο για κάποιον IE ενάντια στον Firefox.

Από το παραπάνω ξεκαθάρισμα προκύπτουν τα εξής ερωτήματα λοιπόν τα οποία θα προσπαθήσω να απαντήσω σε επόμενα θέματα: Πως χειριζόμαστε στο SEO τα IRIs Γιατί ο Firefox δεν μας κάνει την τιμή με τα IRIs Και γενικότερα μια συνολική άποψη και γνώση για τα IRIs
