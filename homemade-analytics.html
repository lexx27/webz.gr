Homemade Analytics!

07 Jul 2008 by PanosJee

Θα θέλατε τα δικά σας analytics να βρίσκονται στο server σας, έχοντας τον απόλυτο έλεγχο των δεδομένων σας και να μπορείτε να γράφετε τα δικά σας modules. Δε συμπαθείτε το Google Analytics ή σας αρέσει να την ψάχνατε λίγο παραπάνω;

Η λύση ακούσει στο όνομα Piwik και πρόκεται για ένα πακέτο Open Source Analytics, θα χρειαστείτε απλά MySQL και PHP για να λανσάρεται τα δικά σας analytics!

To project φαίνεται αρκέτα υποσχόμενο αλλά από ότι φαίνεται επιθυμούν να το κάνουν πιο scalable. Μακάρι να πάει καλά.
