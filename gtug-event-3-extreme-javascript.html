GTUG Event #3, Extreme Javascript

19 Mar 2011 by PanosJee

Extreme Javascript

Ποιο είναι το απόλυτο buzzword της τεχνολογικής σκηνής τον τελευταίο χρόνο; Τι είναι αυτό που άλλοι χαρακτηρίζουν αναγκαίο κακό και άλλοι ως λύση για τα πάντα; Φυσικά δεν είναι άλλο από το παρεξηγημένο Javascript! Είτε είστε υπέρμαχος είτε βλαστημάτε όταν το ακούτε όπως και να έχει θα το χρειαστείτε. Ελάτε στο επόμενο event μας που θα αναλύσουμε προχωρημένα θέματα που έχουν να κάνουν με το omnipresent javascript.

Στα πλαίσια του 3ου GTUG Greece που θα έχει ως κεντρικό του θέμα το Javascript θα μας τιμήσουν με την παρουσία τους οι εξής ομιλήτες:

  • Παναγιώτης Αστήθας Software engineer and explorer

    Scale to infinity with Javascript - An intro to AppengineJS

    Γνωρίζεις javascript και php+mysql. Πασέ, ντεμοντέ και περιορισμένο. Πρέπει να εξυπηρετείστε μιλιούνια επισκεπτών και μισείς το administration; Ε τότε απλά γράψε και το backend σε javascript και πετάξε ψηλά με το Appengine!
  • Χρήστος Στάθης Unbelievably genius software guru

    Intro to GWT (Google Web Toolkit)

    Μισείς το javascript αγαπάς τη Java; Δεν έχεις κανένα λόγο να μην δημιουργείς rich user experiences (in the browser!).
  • Διονύσης Συνοδινός Εditor@InfoQ, Webmaster@NTUA, Author, Open Web Advocate

    Debbugging the Client - Best practices with Google Chrome Tools and Speed Tracer

    Μάθε πώς να βελτιστοποιείς και να λύνςι; τα bugs που έχουν να κάνουν με το Javascript χρησιμοποιώντας τa Chrome Tools. Μη το χάσεις, θα χάσεις!

Οπώς καταλαβαίνεις καλό είναι να έρθεις, Javascript σίγουρα θα αντιμετωπίσεις κάποια στιγμή άρα καλό είναι να μάθεις δίπλα στους pros. Αν πάλι γνωρίζεις ή δε σε ενδιαφέρει το θέμα έλα για να γνωριστούμε και να οργανώσουμε τα μελλοντικά μας events.

Είναι απαραίτητο να δηλώσεις συμμετοχή εδώ γιατί οι θέσεις είναι περιορισμένες.

Το event θα λάβει χώρα στο φιλόξενο CoLab, Ερμού 44, Αθήνα.
