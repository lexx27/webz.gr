Πως να δημιουργείσετε και να διαχειριστείτε τους κωδικούς σας

29 Sep 2007 by lexx

Φαντάζομαι πως όπως και εγώ οι περισσότεροι έχουν δεκάδες για να μην πω εκατοντάδες λογαριασμούς, το οποίο σημαίνει πολλά διαφορετικά passwords. Και για μένα η ασφάλεια των σελίδων μου είναι από τα βασικότερα σημεία της δουλειάς μου γιατί είναι η προστασία των κόπων μου. Ειδικά στους λογαριασμούς οι οποίοι είναι σημαντικοί όπως πχ. paypal, το blog μου, το ftp κλπ. αλλάζω τους κωδικούς κάθε 2 μήνες και θεωρώ ότι είναι κίνηση που μακροπρόθεσμα θα με γλυτώσει από πολλά προβλήματα.

Η ουσία είναι ότι υπάρχουν μικρές πιθανότητες κάποιος να βρει τον κωδικό σας αν είναι σωστά φτιαγμένος και ακόμα μικρότερες αν τον αλλάζετε συχνά, δηλαδή ανν:

  • Έχει τουλάχιστον 8-15 χαρακτήρες ανάλογα την πολυπλοκότητα των χαρακτήρων
  • Περιέχει σύμβολα και αριθμούς πέρα από γράμματα
  • Δεν είναι προφανής όπως το τηλέφωνο ή ημερομηνία γενεθλίων
  • Δεν χρησιμοποιείτε τον ίδιο κωδικό σε πολλούς λογαριασμούς

Όμως αν θεωρήσουμε ότι ακολουθούμε τις παραπάνω συμβουλές τότε είναι πάρα πολύ δύσκολο να έχουμε στην μνήμη μας όλους τους παραπάνω πολύπλοκους κωδικούς που αλλάζουν συνέχεια. Έτσι γίνεται απαραίτητη η χρήση ενός password manager.

Firefox Ο firefox έχει ένα σύστημα αποθήκευση και διαχείριση κωδικών το οποίο απ όσο ξέρω θεωρείται ασφαλές. Οπότε αν δεν υπάρχει μεγάλη ανάγκη για πολλές δυνατότητες είναι μια καλή επιλογή. Αν θέλετε περισσότερα από τον firefox θα τα βρείτε σε σχετικά add-on όπως το Password Hasher.

KeepPass Το KeepPass είναι ένα δωρεάν open-source cross-platform λογισμικό το οποίο τρέχει σε Windows και Linux. Κρατάει σε μια βάση δεδομένων όλους τους κωδικούς και το μόνο που χρειάζεται να θυμάστε είναι το master password. Ένα άλλο εξαιρετικά χρήσιμο χαρακτηριστικό του KeepPass είναι μπορεί να τρέξει και σε usb stick. Έτσι μπορείτε να κουβαλάτε του κωδικούς σας παντού.

Αν και είναι ξεπερασμένα πια τα desktop προγράμματα έχει ακόμα μια ομορφιά

clipperz Για να μη κουβαλάτε όμως το usb stick παντού μπορείτε να επιλέξετε έναν online password manager όπως το clipperz. Είναι εύχρηστο και γρήγορο, σχεδιασμένο για να δέχεται και άλλα όπως πιστωτικές κάρτες, τραπεζικούς λογαριασμούς κλπ. Έτσι όπου υπάρχει πρόσβαση στο διαδίκτυο θα έχετε και τους κωδικούς σας.

Το μόνο που με χάλασε είναι τα πολλά javascript

Php Password Manager Αν από την άλλη έχετε κάποιο βίτσιο όπως εγώ να τα έχετε όλα στον server σας μπορείτε να εγκαταστήσετε το Php Password Manager. Είναι ένα open source php script με όλα τα απαραίτητα στοιχεία.
