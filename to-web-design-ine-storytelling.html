Το Web Design είναι Storytelling

27 Jun 2008 by lexx

Το Web Design μοιάζει πολύ με το Storytelling ορισμένες φορές. Όταν σχεδιάζω μια σελίδα προσπαθώ να σκέφτομαι ότι αφηγούμαι μια ιστορία. Συνήθως προσπαθώ να απευθύνομαι σε ένα παιδάκι. Αν καταφέρει να πιάσει αυτό το νόημα τότε λογικά θα μπορούν όλοι. Και αυτό θα γίνει με έναν πολύ φιλικό τρόπο.

Πέτα την Εισαγωγή

Πόσο βαρετή είναι η εισαγωγή ενός βιβλίου, ή το εισαγωγικό σημείωμα του εκδότη. Ποιος χέστηκε[συγνώμη για την γλώσσα] τι λέει ο κύριος Μπάμπης όταν πας να διαβάσεις Νίτσε;

Όταν διαβάζω ένα μυθιστόρημα κουράζομαι όταν ξεκινάει: "Μια φορά κι έναν καιρό ήταν μια οικογένεια που ζούσε σε ένα σπίτι κάπου στην χώρα της Βαρεμάρας. Είχαν 3 παιδιά, ένα σκυλάκι με κόκκινες βούλες και 3 πόδια το οποίο..."

Μετά κοιμάσαι. Εντάξει το σκυλάκι φαίνεται ενδιαφέρουσα φιγούρα αλλά ας το προσπεράσουμε. Αν ξεκινήσει για παράδειγμα: "Ο Γιώργος πέθανε. Το πτώμα του στο πάτωμα είναι φρέσκο. Όπως και το αίμα στο χαλί. Το μαχαίρι που βρίσκεται δίπλα, με κάνει να αναρωτιέμαι για τον δολοφόνο..."

Το δεύτερο πάει κατευθείαν στο ψητό. Τραβάει την προσοχή, ειδικά σε μια σελίδα που θέλεις να κερδίσεις τον χρήστη σε πολύ λιγότερο χρόνο. Ξεχάστε τα intro pages και τις χαρές και μπείτε κατευθείαν στο ζουμί, σε αυτό που προσφέρει η σελίδα, είτε είναι υπηρεσία, είτε content!

Απλότητα

Μπορεί το πολύπλοκο να εντυπωσιάζει και να αποδίδει κύρος ή και κάτι άλλο. Αλλά αυτό είναι μύθος. Η μαγεία είναι όπως γνωρίζετε να κάνει τα πράγματα όσο πιο απλά για τον χρήστη. Να απαντάς στα πιο βασικά ερωτήματα του μέσα σε λίγα δευτερόλεπτα:

  • Τι πραγματεύεται η σελίδα;
  • Σε ποιους απευθύνεται;
  • Είμαι μέσα σε αυτούς;
  • Τι να κάνω για να πάρω αυτό που μου προσφέρεις;
  • κλπ...

Τα ερωτήματα μπορεί να αλλάζουν ανάλογα τη σελίδα. Αυτές είναι κάποιες σκέψεις που έκανα χθες πριν κοιμηθώ. Θέλω να ακούσω τις ιδέες σας!
