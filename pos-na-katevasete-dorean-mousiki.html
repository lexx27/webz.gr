Πως να κατεβάσετε δωρεάν μουσική

19 Sep 2007 by lexx

Πέρα από τα BitTorrent και τα Peer-to-Peer υπάρχουν και άλλοι τρόποι να αναζητήσετε ή να ανακαλύψετε δωρεάν μουσική. Αρχικά πρέπει να ξέρετε τι ψάχνετε, να έχετε κάνει μια λίστα με καλλιτέχνες για παράδειγμα ώστε να είστε έτοιμοι να τους αναζητήσετε στους παρακάτω προτεινόμενους προορισμούς.

MP3 Blog Aggregators και Μηχανές Αναζήτησης

Skreemr

Το Skreemr πρόκειται για εξαιρετική μηχανή αναζήτησης μουσικής με μεγάλη ποικιλιά. Επιτρέπει το κατέβασμα mp3 και δίνει πολλά χρήσιμα σχετικά link για κάθε τραγούδι για site όπως τα: Amazon, YouTube, Flickr κλπ. Διαθέτει Fifefox Search Plugin για απευθείας αναζήτηση. Επίσης μπορείτε να δείτε τις πιο δημοφιλείς αναζητήσεις της εβδομάδας.

The Hype-Machine

Το Hype-Machine είναι ένας aggregator για μουσικά blog. Αναζητώντας τον καλλιτέχνη και τον τίτλο ενός τραγουδιού θα βρείτε πολλά αποτελέσματα από μουσικά blog τα οποία περιέχουν υλικό για την αναζήτηση σας. Πολλά blog αποθηκεύουν τα mp3 που ανεβάζουν για 1-2 βδομάδες.

SeeqPod

Το SeeqPod είναι μια μηχανή αναζήτησης για mp3. Δοκίμασα και μερικά σπάνια συγκροτήματα και μου έβγαλε αποτελέσματα που δεν περίμενα να βρει. Το κακό είναι ότι δεν αφήνει να γίνει download ένα κομμάτι αλλά μόνο να ακουστεί. Το καλό είναι ότι δίνει τη διεύθυνση του κομματιού, οπότε στην ουσία μπορεί ο χρήστης να κάνει download.

Elbo.ws

Το Elbo.ws είναι παρόμοιο με το Hype-Machine. Κάνεις αναζήτηση και σου προβάλλει blog posts με τους όρους που έθεσες. Στη συνέχεια όμως έχει μια δεξιά στήλη με προτεινόμενα ονόματα για τον όρο που έψαξες. Αν ακολουθήσεις ένα από αυτά μετά σου δίνει να διαλέξει κομμάτια του καλλιτέχνη. Τέλος στου δίνει τα link για τα blog από τα οποία μπορείς να κατεβάσεις αυτά τα κομμάτια. Πολύ καλό.

Project Playlist

To Project Playlist κάνει αναζήτηση στο διαδίκτυο για μουσικά αρχεία. Έχει πολύ μεγάλη ποικιλία και λειτουργεί με ένα πολύ αξιόπιστο σύστημα όπου η κοινότητα κάνει ένα ξεσκαρτάρισμα ανάμεσα στα καλά και στα χαλασμένα αρχεία. Επίσης σου δίνει τη δυνατότητα να δημιουργείς playlists πατώντας add σε κάθε κομμάτι και στην συνέχεια να βάζεις την playlist στο blog σου. Το προτείνω ανεπιφύλαχτα σε όσους ψάχνουν μουσική για το blog τους.

Songza

Στο Songza δυστυχώς δεν δίνεται η δυνατότητα να κατεβάσετε στον υπολογιστή σας τα τραγούδια (μόνο σε iTunes). Μπορείτε όμως να ακούσετε και να δημιουργήσετε λίστες. Υπάρχει μεγάλη ποικιλία και η το UI είναι πολύ λειτουργικό και εύκολο στη χρήση.

Αυτό που μου αρέσει εμένα πιο πολύ όταν ψάχνω μουσική με αυτόν τον τρόπο είναι ότι πέφτω σε πολλά μουσικά blog με πολύ ενδιαφέρον περιεχόμενο. Έτσι γίνεται μια εκπαιδευτική μουσική εξερεύνηση. Καλό ψάξιμο.

Πως να βρείτε ελληνική Μουσική

Πάρα πολλοί αναγνώστες ρωτάνε πως θα βρούνε ελληνική μουσική να κατεβάσουν. Δυστυχώς στα παραπάνω site που παρουσίασα δεν θα βρείτε ελληνική μουσική. Θα πρέπει να βολευτείτε με τα γνωστά BitTorrent και Peer-to-Peer site και software ή με το YouTube.
