FileHo: δωρεάν χωρίς όρια file sharing

18 Sep 2007 by lexx

Μπορώ να ανεβάσω όσα αρχεία θέλω, ότι τύπο θέλω, χωρίς κανένα όριο στο μέγεθος, με όση ταχύτητα έχω και να μείνει uploaded για πάντα; Ναι, μπορείς. Το FileHo είναι μια νέα hosting και sharing υπηρεσία η οποία δίνει όλες αυτές τις δυνατότητες.

Δύο άλλα ενδιαφέροντα στοιχεία είναι η δυνατότητα χρήσης ftp για το ανέβασμα αρχείο. Πράγμα καλό αν κάποιο αρχείο είναι πολύ μεγάλο γιατί προσφέρει μεγαλύτερη σταθερότητα. Επίσης υπάρχει η δυνατότητα να ψάξεις για τα αρχεία τα οποία είναι uploaded με την προϋπόθεση ότι έχει επιλέξει ο χρήστης που τα ανέβασε να είναι εμφανή.
