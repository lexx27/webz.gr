4 Firefox Add-ons που πρέπει να έχει ο κάθε web-designer

23 Sep 2007 by lexx

Ο Mozilla Firefox είναι ο browser του web-designer για ένα πολύ απλό λόγο. Με τα κατάλληλα plugin μετατρέπεται σε ένα ιδανικό εργαστήριο. Προσωπικά δεν μπορώ να δουλέψω χωρίς τον firefox μου ο οποίος είναι κομμένος και ραμμένος πάνω στους ρυθμούς και τον τρόπο εργασίας μου. Θα παρουσιάσω τα plugin που χωρίς αυτά αρνούμαι να δουλέψω και όχι τα plugin που είναι γενικά χρήσιμα.

1. ColorZilla O ColorZilla είναι θυσαυρός για κάθε web και graphic designer. Σου δίνει την δυνατότητα να "αφομοιώσεις" οποιοδήποτε χρώμα στον browser σου και από κει και πέρα να το κάνει copy-paste με τον κωδικό του, να το βάλεις στα αγαπημένα σου ή ακόμα και να κάνεις zoom τη σελίδα για περισσότερη λεπτομέρεια. Πχ. για να πάρεις το χρώμα μιας γραμματοσειράς. Επίσης δίνει την δυνατότητα να δεις σε ποιο class ή id του CSS κοιτάς την κάθε στιγμή.

Θέλω να κλέψω αυτό το μοναδικό πορτοκαλί του webz. Χμ, τόσο εύκολο, τόσο εύκολο πια... :p.

2. IE View Lite

Το IE View Lite είναι μια επανέκδοση του IE View του Paul Roub. Ζυγίζει μονάχα 47.4 KB και έχει τις ίδιες λειτουργίες. Προσωπικά έχω βάλει ένα κουμπάκι στο πάνω μέρος του firefox. Όταν θέλω να δω πως φαίνεται μια σελίδα στον explorer αρκεί να το πατήσω του κουμπί του IE view και μετά να ανάψω ένα τσιγάρο για να μηχανευτώ τρόπους να κάνω τον ie6 να δείχνει τη σελίδα όπως θα έπρεπε. :)

3. MeasureIt

Το MeasureIt όσο απλό κι αν φαίνεται είναι επαναστατικό. Σου δίνει ένα κουτάκι "χάρακα" όπου μετράς αποστάσεις, κουτιά και ότι άλλο θες μέσα σε μια σελίδα σε pixels. Πράγμα που για να το κάνω παλιά έπρεπε να πατήσω Print-Screen και να πάω στο photoshop να μετρήσω.

5616 τετραγωνικά pixel. Αυτό και αν είναι πληροφορία.

4. Greek-English Dictionary Γιατί λεξικό; Δεν φαντάζεστε πόσα ορθογραφικά λάθη βλέπω σε σελίδες. Πάρα μα πάρα πολλά. Ο web-designer οφείλει να παραδίδει μια σωστά ορθογραφημένη σελίδα. Και επειδή δεν είναι φιλόλογος χρειάζεται έναν φίλο. Το αγγλοελληνικό λεξικό του firefox το οποίο θα σας θυμίσει πολύ την λειτουργία του spell checker του word. Και για όσους το έχουν εδώ και δεν το λένε. Ναι, στον firefox γίνεται πλέον μεγάλο μέρος της μετάφρασης μιας πλατφόρμας καθώς χρησιμοποιούν template system και phrase manager. Αφήστε που θεωρώ ότι ο κάθε χρήστης πέρα από τον designer θα πρέπει να το έχει...

Σίγουρα υπάρχουν δεκάδες άλλα addons που βοηθάνε αλλά αυτά είναι τα must!
