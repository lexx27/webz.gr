Windows Server 2008 αέρας OSS

02 Mar 2008 by PanosJee

Αλλάξε ο κόσμος, ω ναι από την ίδια τη Microsoft και μάλιστα από το Open Source Lab της μαθαίνουμε ότι η νέα έκδοση του λειτουργικού έχει επηρεαστεί κατά πολύ από τα μαθήματα της MS από το χώρο του Open Source! Ναι καλά διαβάσατε, όταν πριν λίγα χρόνια η ίδια εταιρία έλεγε ότι το Open Source και ειδικά η άδεια GPL αποτελεί καρκίνωμα για το χώρο του λογισμικού, τώρα επηρεάζεται από αυτό, θέλει να τρέχει πάνω στα προιόντα της και φυσικά να συνεργάζεται με την κοινότητα. Για παράδειγμα η MS έχει καλέσει στα γραφεία της τις ομάδες των Mozilla, Apache, Samba αλλά και συνεργάζεται με τη Zend (call me PHP) και ΧΕΝ και φυσικά βέβαια με τη Novell. It is a cold day in hell όπως θα έλεγαν και οι Αγγλοι...

Ας δούμε μερικές από τις αλλαγές-μαθήματα όπως παρουσιάζονται στον εν λόγω blog

  • Modular architectures, πλέον δε θα βλέπουμε μονολιθικές εκδόσεις προγραμμάτων αλλά μικρές και ευέλικτες αρχιτεκτονικές που επεκτείνονται με modules. Κλασσικό παράδειγμα και πρότυπο ο Apache και ο μαθητής ο ISS 7 που τον ακολουθεί πιστά
  • Programming language agnostic, πλέον κανείς δε σας περιορίζει σε .NET υλοποιήσεις αλλά τρέξτε ότι θέλετε... Θα μου πείτε αυτό γινόταν, αλλά τώρα μπορεί ο ISS να τρέχει μια χαρά PHP και μάλιστα η MS ισχυρίζεται ότι δοκίμασε όλη τη στοίβα εφαρμογών PHP. Ξεκάθαρα θέλει ένα κομμάτι της πίτας της 3ης πιο δημοφιλούς γλώσσας με τη δημιουργία του WISP (Windows - ISS - SQL Server - PHP)
  • Ανάπτυξη βάση σχολίων, πολύ απλά η MS άκουσε τους πελάτες και συνεργάτες τις. Λέτε;;
  • Στοχευμένα συστήματα, αυτό και αν είναι ελπιδοφόρο. Πλέον μπορείτε να στείσετε ένα μηχάνημα windows για να κάνεις ακριβώς αυτό που θέλετε. Τέρμα λοιπόν σε εγκταστάσεις browser, media player και λοιπών βλακειών όταν απλά θέλετε DNS, DHCP, file ή web serving. Απλά επεκτείνετε το Windows Server Core. Μαλίστα στη βασική έκδοση δεν υπάρχει ούτε GUI παρά μόνο command line. Άραγε δικαιούται να ονομάζεται Windows; (!)
  • PowerShell, η νέα command line. Είναι κοινώς γνωστό ότι οι έμπειροι admins θέλουν με scripts να προσαρμόζουν ακριβώς το server ¨οπως αυτοί θέλουν. Τώρα μπορούν και σε Windows.
  • Standards-based communication με τη χρήση CardSpace, Apache’s Axis,Xen virtualization κλπ.
