Mac everywhere

12 Nov 2007 by PanosJee

Η Apple πιάνοντας το νόημα των web-based υπηρεσιών ενσωμάτωσε στο λειτουργικό μια σειρά από υπηρεσίες που δε θα τις λέγαμε καινοτομικές αλλά σαφώς είναι καλά ενσωματωμένες στο Mac ΟS X κάνοντας τες έτσι πιο θελκτικές για τους χρήστες.

Ετσι στο .Mac θα βρούμε μια σειρά υπηρεσιών που προσφέρεται μέσω της υπηρεσίας .Μac (αρχικά διατίθεται δωρεάν με την αγορά υπολογιστή και στη συνέχεια με ετήσια συνδρομή).

  • Web Gallery, απλά το web frontend του iPhoto, όπως το PicasaWeb
  • Website Hosting (τα ευκόλως εννοούμε παραλείπονται)
  • IMAP Mail, προσφέρεται λογαριασμός ΙΜΑΡ μαζί με web frontend
  • Back to My Mac , μια υπηρεσία Remote Control αλά VNC, διαθέσιμη όμως για Mac υπολογιστές
  • Sync, συχρονισμός ημερολογίων, αλληλογραφίας, επαφών...
  • iDisk, αποθηκευτικός χώρος στο web
  • Groups, social network για mac users
  • Backup, backup :)
  • και όλα αυτά με 10GB
Από ότι φαίνεται όλα αυτά έχουν και καλή υποστήριξη για το iPhone αφού φυσικά η Apple έχει τον απόλυτο έλεγχο γύρω από το οικοσύστημα της χωρίς να υποστηρίζει ιδιαίτερα κάθε τι άλλο.

Βέβαια και η Microsoft έχει τις Live υπηρεσίες τις αλλά προς το παρόν και στα ΧΡ που έχω δεν είναι τόσο συνδεδεμένες με το λειτουργικό σύστημα, αν και τα Vista είναι πιο web friendly το project όμως που αναμένεται να δώσει μεγάλη ώθηση σε αυτήν τη φιλοσοφία ενοποίησης web και desktop είναι το Gnome Online Desktop
