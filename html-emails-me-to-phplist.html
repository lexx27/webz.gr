HTML Emails με το phpList

07 Jul 2008 by lexx

Το phpList είναι μια Open Source πλατφόρμα για Newsletters. Το χρησιμοποίησα πριν ένα μήνα για έναν πελάτη και είμαι πάρα πολύ ευχαριστημένος από το αποτέλεσμα. Είναι λίγο δύσκολο στην χρήση του αλλά δίνει πάρα πολλές δυνατότητες όπως:

  • HTML Emails με Templates
  • CVS Import ή Database Table Import.
  • Email lists Groups
  • RSS Feeds

Και η λίστα συνεχίζει όπως θα δείτε και στην επίσημη σελίδα.

Όταν λέω δύσκολο εννοώ ότι έχει πάρα πολλές επιλογές για να χωρέσει τα πάντα. Επίσης έχει τα κλασικά fields που έχουν πολλά admin που πρέπει να φας τον κόσμο για να καταλάβεις τι σημαίνουν ή να τα αφήσεις κενά και να προσευχηθείς.

Κατά τα άλλα δουλεύει ολόσωστα και ότι ήθελα να πετύχω το πέτυχα. Δυστυχώς όμως είναι ένα εργαλείο που ο πελάτης δεν θα μπορεί να χειριστεί μόνος του, οπότε μην ελπίζεται ότι μπορείτε να προσφέρετε αυτονομία με αυτό.

Τα HTML Emails είναι βάσανο

Νομίζω ότι δεν θα διαφωνήσει κανείς πως η σχεδίαση των HTML email είναι σκέτο βάσανο και καθόλου διασκεδαστική διαδικασία. Είναι αρκετά δύσκολο να τα κάνεις να φαίνονται σωστά σε όλες τις υπηρεσίες και τους Client. Χρειάζονται ειδική μεταχείριση.
