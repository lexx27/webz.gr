Web 2.0 Domain Name Generator

17 Aug 2007 by lexx

Ένα εξαιρετικά χρήσιμο domain generator μου είχε προτείνει ο Γιάννης όταν έψαχνα όνομα για ένα site. To Dotomator για το οποίο έχω την εντύπωση ότι οι δημιουργοί του βρήκαν το όνομα από το ίδιο τους το script. :p

Το Dotomator είναι ένα απλό site χωρισμένο σε 3 κατηγορίες:

Dot-o-mator: Βάζεις έναν αριθμό λέξεων όπου χαρακτηρίζουν τη σελίδα σου και αυτό αναλαμβάνει να τα συνδυάσει και να παράγει κάποια ωραία web 2.0 ονόματα.

Web 2.0 Name Generator: Πατάς generate name και αρχίζει να βγάζει ονόματα... Domain Naming Tips: Συμβουλές για τη σωστή αναζήτηση και επιλογή domain.

Από την χρήση του Dotomator δεν βρήκα απευθείας όνομα, όμως μου έδωσε έμπνευση και με βοήθησε να σκεφτώ με περισσότερη ποικιλία.
