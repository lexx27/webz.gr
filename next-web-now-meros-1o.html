Next Web Now (μέρος 1ο)

14 Dec 2007 by PanosJee

Μόλις γύρισα από το event της Ελληνικής Microsoft που ως συνήθως ήταν αρκετά ενδιαφέρον (αν και δε χρησιμοποιώ ιδιαίτερα τις τεχνολογίες της όσο αφορά το development - deployment) με πολύ καλούς ομιλητές και previews επερχόμενων τεχνολογιών. Συνοπτικά παρουσιάστηκαν γενικά και θεωρητικά θέματα για το τωρινό και μελλοντικό web, μια συνοπτική παρουσίαση της νέας οικογένειας εργαλείων σχεδίασης Expression, αρκετός λόγος για το Silverlight καθώς και μια σειρά υπηρεσιών Live.

Next.Web.Now

Πιο αναλυτικά στο πρώτο session ο Παναγιώτης Κοντόπουλος (με χαρά έμαθα ότι είναι και αναγνώστης του webz) μας παρουσίασε τη σημερινή κατάσταση στο web όχι απλά μιλώντας γενικά περί Web 2.0 αλλά παρουσιάζοντας το web σαν πλατφόρμα αλλά και μάλιστα παρουσιάζοντας μια διαστρωματική προσέγγιση σε αυτό τύπου OSI. Όταν ο Παναγιώτης ανεβάσει τη διαφάνεια θα σας ενημερώσω σχετικά. Τέλος μας έκανε ένα demo του Popfly, ένα mashup editor for the mashes.

Creating Rich User Experience

Ο Μάνος Κελαϊδίτης μας παρουσίασε κομμάτια της σουίτας Expression που απευθύνεται σε designers και συνεργάζεται άψογα με το Visual Studio. Τα στοιχεία που κράτησα είναι το γεγονός ότι ακόμα και γραφικά αποθηκεύονται ως αρχεία XAML, μια διάλεκτος XML που καταλαβαίνει το Windows Presentation Foundation του .ΝΕΤ 3.0 και επίσης ότι πλέον η Microsoft δίνει τεράστιο βάρος στο user experience ακόμα και τυπικών παραθυρικών προγραμμάτων όπως π.χ. ένα λογιστικό πρόγραμμα. Είμαι ιδιαίτερα υπέρ αυτής της εξέλιξης, και από ότι φαίνεται από web designers θα πάμε και σε Windows application designers.

Reaching Audiences with Services

O Κώστας Πάντος μας παρουσίασε εκτενώς την υπηρεσία Silverlight Streaming, το Silverlight είναι στην ουσία ένα wannabe flash killer αλλά νομίζω πώς όντως είναι πιο πλούσιο και πιο ανοιχτό, στην ουσία το Silverlight Streaming επιτρέπει να ανεβάζουμε αρχεία encoded σε Silverlight format στους servers της Microsoft και στην συνέχεια από εκεί να υπάρχει πραγματικό streaming και παρουσιάζουμε απλά τα videos μέσα σε ένα iframe, το αρνητικό της υπόθεσης είναι ότι το αρχείο πρέπει να είναι ήδη στο format αυτό. Στη συνέχεια παρουσιάστηκαν μια σειρά από πραγματικά ενδιαφέροντες υπηρεσίες Live:

  • Photos
  • Virtual Earth
  • Live ID, κάτι άκουσα για OpenID συμβατότητα
  • Messenger

Πολύ ενδιαφέρον είναι το photos αφού μπορείς να το χρησιμοποιήσεις και σαν CDN. Γενικά η Microsoft από ότι καταλαβαίνω προσπαθεί να είναι καλή, ανοιχτή, συμβατή με τα πρότυπα, το καλό Silverlight και το κακό proprietary format... Άλλαξε κάτι ξαφνικά; Απλά η Microsoft έχει μείνει αρκετά πίσω στο web σε διαφημιστικά έσοδα και βάζει τις βάσεις για το μέλλον. Να με θυμηθείτε ότι κάποτε τα Windows θα είναι δωρεάν... Αν δείτε σχεδόν οι περισσότερες υπηρεσίες είναι δωρεάν αλλά μετά από ένα σημείο views, requests ή θα ζητά λεφτά ή θα δείχνει διαφημίσεις και η MS καταλαβαίνει ότι εκεί είναι το ψωμί.
