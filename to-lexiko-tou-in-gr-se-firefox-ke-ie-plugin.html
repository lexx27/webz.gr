Το λεξικό του In.gr σε Firefox και IE (Plugin)

05 Jun 2008 by lexx

Ένα εργαλείο που χρησιμοποιώ πάρα πολύ συχνά είναι το λεξικό του In.gr. Όμως για να το φτάσω θα πρέπει να το έχω κάπου bookmark και αυτό με κουράζει. Αναρωτήθηκα λοιπόν πριν λίγο: Δεν θα ήταν καλή ιδέα να υπάρχει στα search engines του Firefox ή του Internet Explorer πάνω δεξιά; Έτσι έψαξα στα Search Engines Add-ons του Firefox. Δεν βρήκα κάτι και προωθήθηκα στο SearchPlugins.net το οποίο έχει όπως καταλαβαίνετε και από τον τίτλο plugins για μηχανές αναζήτησης. Έκανα και εκεί μια αναζήτηση για να βρω μήπως υπάρχει ένα plugin για το λεξικό του in.gr και όντως υπήρχε. Το εγκατέστησα αλλά αντιμετώπιζε πρόβλημα με τα ελληνικά. Έτσι χρησιμοποιώντας το Plugin Generator δημιούργησα ένα καινούργιο που δουλεύει σωστά. Install It Update: Υπάρχει και ένα πολύ χρήσιμο plugin για firefox όπως με ενημέρωσε ο Vasiles, το οποίο μεταφράζει μια λέξη την οποία θα επιλέξετε. με δεξί κλικ.
