Rails 2.0

12 Dec 2007 by PanosJee

Κυκλοφόρησε η νέα έκδοση του δημοφιλούς framework που πλέον έχει μπει για τα καλά στη φάση της ωριμότητας ή τουλάχιστον της πρώιμης αλλά σίγουρα έχει περάσει την εφηβεία του.

Τι νέο υπάρχει; Καταρχήν είναι μικρότερο με αρκετές λειτουργίες να φεύγουν από το core και να γίνονται plugins. Μια πολύ σημαντική για μένα αλλαγή είναι η δυνατότητα φόρτωση διαφορετικών templates π.χ. ανάλογα με τον client μπορείτε να φορτώσετε διαφορετικό template έτσι ένα κοινός browser και το Iphone θα έχουν διαφορετική εμφάνιση. Μια μικρή λίστα με αλλαγές που μου έκαναν εντύπωση

  • Βελτιωμένη υποστήριξη RESTful urls και δυνατότητα δημιουργίας meaningful urls όπως admin_products_list
  • Multiview, όπως είπα για τις πολλαπλές όψεις με την εισαγωγή ενός μικρού φίλτρου
  • Κάθε αντικείμενο είναι και ένα URL !! π.χ. redirect_to(person)
    link_to(person.name, person)
    form_for(person)
  • Διαχείριση sessions, επιτέλους ! Όλα τα sessions πλέον αποθηκεύονται σε cookies και έχει σε φάκελο στον server
  • Κατασκευή Atom feed on the fly
  • Query Cache, δηλαδή το ίδιο query ανά action δεν επαναλαμβάνεται αλλά τα δεδομένα από την cache χρησιμοποιούνται
