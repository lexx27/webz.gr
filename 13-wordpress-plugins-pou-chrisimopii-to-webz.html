13 Wordpress Plugins που χρησιμοποιεί το Webz

21 Jan 2009 by lexx

Τα Wordpress Plugins Lists  είναι κάτι το οποίο εθίζει και πορώνει. Ούτε που θυμάμαι πόσα άρθρα με τίτλο "10 must-have Wordpress Plugins" έχω ανοίξει. Και τώρα αν έβλεπα ένα θα το άνοιγα να το διαβάσω ακόμα και αν ήξερα ότι και τα δέκα τα έχω εγκατεστημένα. Είναι ψυχολογικό. I can't help it. Παρακάτω λοιπόν θα γράψω μερικά από τα Plugins που χρησιμοποιεί το Webz:

All in One SEO Pack

Αλήθεια σας λέω, δεν το χρησιμοποιώ. Κάνει κάποια πράγματα μόνο του και αυτά με ενδιαφέρουν. Πάντως δίνει πολλές επιλογές για την παραμετροποίηση των meta κάθε post και κάθε σελίδας. Όρεξη να έχεις.

Dofollow

Είναι ένας γρήγορος τρόπος να βγάλεις το nofollow από τα σχόλια. Περισσότερα για τους λόγους που μπορεί κάποιος να θελήσει να κάνει κάτι τέτοιο σε παλαιότερο άρθρο του Webz.

Google XML Sitemaps

Βασικό plugin το οποίο βοηθάει σημαντικά στο SEO της σελίδας καθώς ενημερώνει το Google για τα νέα post του Wordpress σας.

LifeStream

Το πιο καλόφτιαγμένο plugin για Wordpress που παρουσιάζει το Social Lifestream σας.

Post-Plugin Library

Είναι ένας πολύ απλός και παραμετροποιήσημος τρόπος να εισάγεται τα plugins Similar Posts, Recent Posts, Random Posts, και Recent Comments στη σελίδα σας.

Simple Tags

Παλαιότερα που το Wordpress δεν είχε πολύ καλό tag management ήταν εξαιρετικά χρήσιμο. Ακόμα βολεύει γιατί δίνει προτάσεις για tag σε κάθε post.

Subscribe2

Μια πολύ καλή λύση αν θέλετε να δίνεται την δυνατότητα στους αναγνώστες σας να λαμβάνουν ενημερώσεις για νέα θέματα μέσω email.

Subscribe To Comments

Απαραίτητο Plugin το οποίο ενημερώνει τον επισκέπτη μέσω email, στα θέματα που συμμετέχει, αν υπάρχει νέο σχόλιο.

Theme Tester

Μπορείτε να αλλάζετε το Theme σας χωρίς να παίρνουν χαμπάρι οι αναγνώστες. Το κακό είναι ότι θα έχετε προβληματάκι αν χρησιμοποιείτε Widget.

WordPress.com Stats

Δεν είναι και το καλύτερο σύστημα για στατιστικά. Είναι όμως γρήγορο και εύχρηστο για κάποια καθημερινά stats που μας ενδιαφέρουν και είναι hosted στο Wordpress.com.

WordPress Database Backup

Το καλύτερο Feature είναι τα προγραμματισμένα Backup τα οποία αποστέλλονται με email.

WP-Print

Πάρα πολύ καλά φτιαγμένο Plugin. Δώστε ειδική έκδοση της κάθε σελίδας του site σας για εκτύπωση. Δυστυχώς κάθε φορά που κάνει update το Wordpress δεν δουλεύει. Έχει bugs με το URL Rewrite.

WP-Syntax

Συμπαθητικό plugin για να δείχνετε κώδικα με highlight. Γενικά δεν είμαι από κανένα ευχαριστημένος. Αυτό είναι το λιγότερο κακό.
