Grid Hosting

05 Nov 2007 by PanosJee

Ναι λοιπόν τώρα grid services για όλους! Τι είναι όμως grid computing? Από τη wikipedia μπορούμε να αλιεύσουμε την απάντηση, ελληνιστί λέγεται "υπολογισμός πλέγματος", σε απλά ελληνικά μιλάμε για ένα δίκτυο υπολογιστών όπου οι διεργασίες δεν χρειάζεται να τρέχουν σε ένα από τα μηχανήματα του δικτύου αλλά μπορούν είτε να διαμοιράζονται σε όλα τα μηχανήματα ή αν κάποιο μηχάνημα είναι φορτωμένο η διεργασία να εκτελεστεί σε άλλο χωρίς να πάρουμε το πάρουμε χαμπάρι... Έτσι επιτυγχάνεται μια τρομερή οικονομία σε πόρους αλλά κατά κύριο λόγω μπορούμε σχεδόν πάντα να αντιμετωπίζουμε τα load traffics... Ακόμα αν κάποιο μηχάνημα καταρρεύσει κάποιο άλλο θα συνεχίσει τη δουλειά του χωρίς καμία απώλεια.

Σε αυτές τις υλοποιήσεις για web hosting τα δεδομένα αποθηκεύονται σε ένα παρόμοιο δίκτυο βάσεων δεδομένων που ονομάζεται SAN (Storage Area Network). Γενικότερα αυτή η χρήση του grid για hosting προσφέρει πολύ μεγάλη διαθεσιμότητα στις δικτυακές εφαρμογές. Δεδομένου ότι πάρα πολλοί servers έχουν τεράστια idle times πολλοί πίστεψαν ότι το virtualisation έδινε μια λύση και ότι ο καθένας θα είναι χαρούμενος με το private virtual server (προσωπικά είμαι γιατί και root είμαι αλλά και support έχω). Όμως φανταστείτε τη δύναμη που θα είχατε με μια grid εφαρμογή σε ένα τεράστιο data center.
