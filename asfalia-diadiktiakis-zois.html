Ασφάλεια διαδικτυακής ζωής

14 Dec 2007 by lexx

Θα ήθελα μια ασφάλεια ζωής για 5 blog, γύρω στα 90 profiles. εεε; όχι το myspace δεν το θέλω, ας χαθεί, έχει ήδη πολύ spam :p

Ένα πάρα πολύ ωραίο και φρικαλέο θέμα πραγματεύεται ο nikan στο blog του για την ψηφιακή μας ζωή όταν θα αποχωρήσουμε από τον πλανήτη γη.

Πολύ ωραίο θέμα! Δεν ξέρω κατά πόσο ο υπεύθυνος θα πρέπει να είναι το κράτος κάθε πολίτη ξεχωριστά. Σε πολλές περιπτώσεις όπου ο συγγραφέας είναι ανώνυμος και δεν μπορούμε να ξέρουμε την εθνικότητα του τότε χάνεται; Μια παγκόσμια διαδικτυακή αρχή ή υπηρεσία ίσως να λειτουργούσε καλύτερα. Το αρχή δεν μου αρέσει. Νομίζω ότι κάτι σαν το internet archive σε μια πιο ολοκληρωμένη και αξιόπιστη μορφή ότι θα ήταν πολύ χρήσιμο.

Στην ουσία τα δυναμικά site όπως τα blog θα μετατρέπονται σε στατικά και θα αποθηκεύονται σε server όπως ακριβώς ήταν παλιά. Χωρίς όμως την δυνατότητα να προστεθούν σχόλια ή να επεξεργαστούν. Θα γίνονται δηλαδή όλα html και css. Δεν θα υπάρχει πάρε δώσε με κάποια βάση δεδομένων για οικονομία και ασφάλεια.

Αυτό σημαίνει όμως ότι κάποιος θα πρέπει να καταβάλει χρόνο και να επενδύσει σε server για να κάνει κάτι τέτοιο. Αρχίζουμε λοιπόν και μιλάμε για μια υπηρεσία με μη κερδοσκοπικό χαρακτήρα που θα ζει από δωρεές όπως η wikipedia για παράδειγμα.

Ή, κάτι πιο τρελό! Ίσως μπορούμε να μιλάμε για ασφάλεια διαδικτυακής ζωής. Ιδιωτικές υπηρεσίες τις οποίες θα πληρώνεις όσο ζεις και θα τους δίνεις τους κωδικούς σου έτσι ώστε όταν αποβιώσεις να χειριστούν το αρχείο σου με τον τρόπο που επιθυμείς. Φρικτό ε; :)
