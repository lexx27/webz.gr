Διαγωνισμός: Κερδίστε Web Hosting στην CretaForce

05 Jul 2009 by lexx

Για να συμμετέχετε στον διαγωνισμό απλά απαντήστε σε αυτό το post μέχρι και τις 31 Ιουλίου. Στο διαγωνισμό δε συμμετέχω εγώ, οι συντάκτες του webz.gr, όσοι έχουν εταιρείες παροχής υπηρεσιών internet και οι εργαζόμενοι στην CretaForce.

Όσοι δηλώσουν μέρος στο διαγωνισμό αλλά και όσοι επιθυμούν μετά το πέρας του και μέχρι το τέλος του Αυγούστου να  απαντήσουν, θα μπορούν να παραγγείλουν οποιοδήποτε πακέτο hosting επιθυμούν με 20% έκπτωση αρκεί μετά τη παραγγελία τους να στείλουν μήνυμα στο sales [at] cretaforce.gr που να αναφέρουν τη συμμετοχή τους στο διαγωνισμό.

O νικητής θα κερδίσει ένα πακέτο φιλοξενίας για 1 χρόνο που περιέχει:

Πακέτο: + Χώρος 1GB + Μεταφορά 50GB

Λογισμικό: + FreeBSD + Lighttpd + PHP + MySQL + CretaPanel

Χαρακτηριστικά: + Απεριόριστα Domains + Απεριόριστα Subdomains + Απεριόριστα E-mails + Απεριόριστες βάσεις δεδομένων + Απεριόριστα FTP accounts

Αντίγραφα ασφαλείας: + Ημερήσια και εβδομαδιαία αντίγραφα ασφαλείας + Εβδομαδιαία απομακρυσμένα αντίγραφα ασφαλείας σε 2 servers + Πρόσβαση στα backups με χρήση ssh2 και ftp

Υποστήριξη: + 24ωρη τεχνική υποστήριξη + Δωρεάν εγκατάσταση διάφορων scripts + Δωρεάν επαναφορά από τα αντίγραφα ασφαλείας
