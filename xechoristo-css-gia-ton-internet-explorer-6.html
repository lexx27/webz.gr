Ξεχωριστό CSS για τον Internet Explorer 6

03 Oct 2007 by lexx

Την βδομάδα που πέρασε είχα αναλάβει ένα site για joomla. Φτιάχνοντας το CSS του theme αντιμετώπισα πάρα πολλά προβλήματα με τον ie 6. Όταν έφτιαχνα κάτι εκεί χάλαγε στον firefox. Όταν πλέον πήρα απόφαση ότι δεν υπάρχει περίπτωση να συντονίσω τον ie με τους άλλους φυσιολογικούς browser προχώρησα στη δημιουργία ενός ξεχωριστού CSS για αυτόν.

Αυτή η τεχνική δεν είναι πολύ βολική εάν γίνονται συχνά αλλαγές στο site αλλά αν πρόκειται το theme να μείνει για καιρό θεωρώ πολύ πιο πρακτικό να γίνει ξεχωριστό CSS αρχείο για τον ie. Η μέθοδος για να γίνει αυτό είναι πολύ απλή. Μέσα στο <head> της σελίδας προσθέτουμε τα παρακάτω:

Για όποιον browser έχει την τύχη να μην είναι ο ie 6

<!--[if !IE]>
<link rel="stylesheet" type="text/css" href="style.css" />
<![endif]-->

Για τον ie6

<!--[if IE 6]>
<link rel="stylesheet" type="text/css" href="style_ie6.css" />
<![endif]-->

Για τον ie6 και τις χαμηλότερες εκδόσεις

<!--[if lte IE 7]>
<link rel="stylesheet" type="text/css" href="style_ie6_and_down.css" />
<![endif]-->
