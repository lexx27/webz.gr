Πως φαίνεται μια σελίδα σε όλους τους browser

23 Sep 2007 by lexx

Είναι σχεδόν αδύνατο αλλά σημαντικότατο να ξέρουμε πως φαίνεται η σελίδα μας σε όλους τους browser. Πως μπορούμε όμως να το πετύχουμε αυτό τη στιγμή που θα πρεπε να έχουμε πρόσβαση, σε διαφορετικά συστήματα, με διαφορετικούς browser και ρυθμίσεις. Την λύση δίνει το Βrowsershots για το οποίο διάβασα πριν λίγο και τελικά δοκίμασα.

Μπαίνοντας στο Βrowsershots υποβάλεις την διεύθυνση της σελίδας σου στη σχετική φόρμα και επιλέγεις τους browser που θέλεις να γίνει το test. Υποστηρίζει browser για Linux, Windows και Mac. Πέρα λοιπόν από τους browser επιλέγεις προαιρετικά και Screen size, Color depth και έκδοση για Javascript , Java, Flash.

Από τη στιγμή που υποβάλεις την σελίδα σου περνάει κανένα εικοσάλεπτο για να βγουν οι πρώτες φωτογραφίες. Αν θέλετε μπορείτε να δείτε πως τα πάει το webz ως τώρα για να πάρετε και μια γεύση της υπηρεσίας. Δεν έχω καταλάβει αν αυτή η σελίδα είναι προσωρινή ή μόνιμη, το καλό είναι πάντως ότι δίνει την δυνατότητα για download των screenshots.

Αν σας ενδιαφέρει μπορείτε να συνεισφέρετε και στην ελληνική μετάφραση του project η οποία έχει μπει στη λίστα αλλά με progress 0.
