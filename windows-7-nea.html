Windows 7 νέα

28 May 2008 by PanosJee

Μιας και κανείς μας δεν πολυχαίρεται με τα Windows Vista ας πάρουμε μια άποψη από τα επερχόμενα Windows. Καταρχήν όπως διαβάζουμε ο πυρήνας δε θα είναι κάτι ριζικά νέο αλλά μια μετεξέλιξη αυτού των Windows 2008 που αποτελεί μετεξέλιξη αυτού των Vista. H Microsoft ελπίζει ότι η νέα έκδοση του λειτουργικού δε θα απαιτεί περισσότερους πόρους αλλά και θα είναι συμβατή με τους drivers και τα προγράμματα της τωρινής έκδοσης. Επίσης δίνουν βάρος στην αφή, ναι ξέρω τι θυμίζει αλλά ας το δούμε Video: Multi-Touch in Windows 7
