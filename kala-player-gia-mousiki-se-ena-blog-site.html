Καλά Player για Mουσική σε ένα Blog/Site

07 Oct 2006 by lexx

updated: 22-9-2008

Είναι τρομερά ενοχλητικό να μπαίνεις σε ένα blog και να παίζει μουσική αυτόματα. Οι λόγοι είναι πολλοί. Μπορεί να είμαι στο γραφείο, μπορεί να ακούω ήδη μουσική και ξαφνικά να νιώσω ότι είμαι σε καλοκαιρινό bar που ακούγεται η μουσική του και μπλέκεται με των γειτονικών, μπορεί απλά να έχω νεύρα. Γι αυτό θα σας προτείνω κάποιες εναλλακτικές λύσεις.

Το θέμα λοιπόν είναι ότι αν έχει κάποιος απαιτήσεις από τον player που χρησιμοποιεί όπως για παράδειγμα να μην παίζει αυτόματα, να γράφει πάνω τα τραγούδια κλπ., θα πρέπει να χρησιμοποιήσει ένα πιο «δυνατό» λογισμικό από αυτά που κυκλοφορούν στα περισσότερα blog.

Εγώ χρησιμοποιώ όταν χρειαστεί το Flash Mp3 Player του jeroenwijering

Θεωρητικά πρέπει να έχετε άδεια για να παίζεται εμπορική μουσική. Πρακτικά όμως δεν το σέβεται κανείς αυτό οπότε πάμε στο ψητό! Για τη νόμιμη και ηθική χρήση μουσικής σε blog και για τις σχετικές υπηρεσίες θα μιλήσουμε σε άλλο ποστ [αν δεν το έχω κάνει ήδη]

Παρακάτω δεν είναι αναλυτικός οδηγός απλά μια ιδέα για την όλη λειτουργία.

Απαιτήσεις;

Πρέπει να έχετε hosting για να εγκαταστήσετε τους player αλλά και για να ανεβάσετε τα mp3 που θέλετε να παίζουν στο blog σας. Υπάρχουν εκατομμύρια από αυτούς και είναι πάρα πολύ φθηνοί.

Πως λειτουργούν αυτοί οι player;

Η λειτουργία τους είναι απλή και εξαιρετικά εύχρηστη. Από όλα τα αρχεία 2 σας ενδιαφέρουν τα οποία είναι τύπου XML. Το config και το playlist. Και εδώ ακριβώς είναι όλα τα λεφτά.

config: Μέσα στο config βρίσκονται όλες οι ρυθμίσεις του player. Για παράδειγμα το μέγεθος, τα χρώματα, αν θα φαίνεται η playlist, αν θα ξεκινά αυτόματα κλπ.

playlist: Εδώ καθορίζεται ποια τραγούδια θα παίζουν, τα ονόματα τους, της σειρά και άλλες τυχόν πληροφορίες που θα θέλετε να βλέπει ο επισκέπτης.

Τα καλά νέα είναι ότι μπορείτε να φτιάξετε όσες playlist θέλετε και όσα config θέλετε. Με λίγα λόγια μπορείτε να έχετε πολύ εύκολα όσους player σαν κάνει κέφι και με την προσθήκη ενός απλού κώδικα να τους ανεβάζετε σε όποιο site έχετε πρόσβαση στον κώδικα.

Ένα άλλο θετικό στοιχείο είναι ότι μπορείτε να δίνετε τον κώδικα και σε άλλους και να έχουν τον player σας στο blog τους για παράδειγμα.

Εγκατάσταση

Η εγκατάσταση είναι απλή και παρόμοια και στους δύο. Κατεβάζετε το zip αρχείο, αντιγράφετε τα αρχεία στο ftp σας και ορίζετε το config. Δεν θα γράψω περισσότερα. Αν τελικά διαλέξετε να εγκαταστήσετε ένα από τα δύο θα βρείτε αναλυτικούς οδηγούς στα site τους.

Οι player είναι δωρεάν και το μόνο που χρειάζεται είναι ένα αστείο ποσό για hosting. Περιμένω τις απορίες σας, αν υπάρχουν.
