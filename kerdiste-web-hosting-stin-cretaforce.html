Κερδίστε Web Hosting στην CretaForce

25 Dec 2008 by lexx

Κερδίστε το web hosting σας για ένα χρόνο από την CretaForce Internet Services, μια εταιρεία που δε χρειάζεται συστάσεις.

Για να συμμετέχετε στον διαγωνισμό απλά απαντήστε σε αυτό το μήνυμα έως τα μεσάνυχτα της Τετάρτης 31/12/2008 (με το ρολόι του webz.gr). Στο διαγωνισμό δε συμμετέχω εγώ, όσοι έχουν εταιρείες παροχής υπηρεσιών internet και οι εργαζόμενοι στην εταιρία CretaForce Internet Services.

Επίσης όσοι δηλώσουν μέρος στο διαγωνισμό και μέχρι το τέλος του Ιανουαρίου θα μπορούν να παραγγείλουν οποιοδήποτε πακέτο hosting επιθυμούν με 20% έκπτωση αρκεί στο πεδίο Κουπόνι να γράψουν τη λέξη webz.gr και για e-mail να δηλώσουν αυτό που χρησιμοποίησαν για να πάρουν μέρος στο διαγωνισμό.

Η κλήρωση θα πραγματοποιηθεί την Πέμπτη 1/1/2009 και ο νικητής θα κερδίσει ένα πακέτο φιλοξενίας που περιέχει: Χώρος: 1GB Μεταφορά: 50GB

Λογισμικό: + FreeBSD 7.0 + Lighttpd 1.4.20 + PHP 5.2.8 + MYSQL 5.0.67 + CretaPanel

Χαρακτηριστικά: + Απεριόριστα Domains + Απεριόριστα Subdomains + Απεριόριστα E-mails + Απεριόριστες βάσεις δεδομένων + Απεριόριστα FTP accounts

Αντίγραφα ασφαλείας: + Ημερίσια και εβδομαδαία αντίγραφα ασφαλείας + Εβδομαδιαία απομακρυσμένα αντίγραφα ασφαλείας σε 2 servers + Πρόσβαση στα backups με χρήση ssh2 και ftp

Υποστήριξη: + 24ωρη τεχνική υποστήριξη + Δωρεάν εγκατάσταση διάφορων scripts + Δωρεάν επαναφορές από τα αντίγραφα ασφαλείας
