Ο γρηγορότερος τρόπος να φτιάξει ο οποιοσδήποτε μια σελίδα

30 Sep 2007 by lexx

Δεν θα ήταν πολύ χρήσιμο μέσα σε λίγα δευτερόλεπτα να μπορείς να φτιάξεις μια γρήγορη html σελίδα και να την δημοσιεύσεις στο διαδίκτυο; Μπορώ να φανταστώ έναν αριθμό λόγων για κάτι τέτοιο. Είναι χρήσιμη υπηρεσία:

  • Για κάποιον που δεν έχει σελίδα και θέλει να αναρτήσει πληροφορίες και link για κάποιο θέμα γρήγορα
  • Για κάποιον που θέλει ανώνυμα να προωθήσει κάτι
  • Για συνεργατική ανάπτυξη κειμένων
  • Για γρήγορο preview html κώδικα
  • Για οποιονδήποτε άλλο λόγο δεν υπάρχει χρόνος ή διάθεση δημιουργίας λογαριασμών, blog κλπ.

Όλα αυτά τα λέω γιατί ανακάλυψα το <a href=""http://www.jottit.com/">Jottit στο οποίο κυριολεκτικά μέσα σε δευτερόλεπτα φτιάχνεις σελίδες με html χωρίς την ανάγκη καν να γίνεις μέλος. Είναι πολύ απλή και απέριττη υπηρεσία. Σου δίνει την δυνατότητα να κάνεις claim μια σελίδας βάζοντας κωδικό και αποστέλλοντας στο email σου τα σχετικά δικαιώματα.

Θα μπορούσε να γίνει γρηγορότερα;

Οι επιλογές που δίνει είναι λίγες και βασικές. Από πλευράς λειτουργιών υπάρχει η επεξεργασία του html κώδικα, permissions, το όνομα και το domain της σελίδας. Επίσης υπάρχει και η δυνατότητα feed εξαγωγής. Από πλευράς εμφάνισης δίνονται κάποια βασικά εργαλεία αλλαγής χρωμάτων και όχι κάτι παραπάνω. Όπως είναι λογικό δεν χρειάζεται κάτι παραπάνω καθώς σκοπός της σελίδας δεν είναι η δημιουργία site.

Τα διακαιώματα της κάθε σελίδας μπορούν να σας δώσουν και ιδέες για περισσότερες χρήσεις. Μπορεί η σελίδα να είναι κρυφή και να χρειάζεται κωδικός για προβολή και επεξεργασία. Μπορεί να είναι δημόσια και να χρειάζεται κωδικός μόνο για την επεξεργασία. Και τέλος μπορεί να είναι και ανοιχτή. Δηλαδή να μπορούν να την δουν και να την επεξεργαστούν όλοι.

Για να δείτε ένα παράδειγμα έφτιαξα μια public σελίδα στην οποία αντέγραψα το html της φάρμας του webz έτσι για να δω πως λειτουργεί. Έφτιαξα επίσης μια open σελίδα όπου μπορείτε να δείτε πως δουλεύει και να την μεταχειριστείτε όπως θέλετε.
