Iterasi: Αποθηκεύστε το διαδίκτυο για πάντα

08 Jan 2009 by lexx

Πριν από καιρό σκεφτόμουν ότι θα ήθελα κάτι σαν το delicious το οποίο να μπορεί να αποθηκεύει ολόκληρη τη σελίδα όπως είναι και να την κρατάει αναλλοίωτη στο χρόνο. Βέβαια ήταν απλά μια ιδέα, από αυτές που σημειώνεις κάπου και μια μέρα την βλέπεις υλοποιημένη από κάποιον και πραγματικά χαίρεσαι γιατί είναι σαν κάποιος να πραγματοποιεί τις φαντασιώσεις σου.

Όλα τα παραπάνω λοιπόν με αφορμή το Iterasi, αυτή η υπηρεσία αν μπορούσαμε να την ορίσουμε κάπως θα λέγαμε πως είναι η social εκδοχή του Wayback Machine (το οποίο δεν λειτουργεί και τόσο καλά και φαίνεται να έχει μείνει πίσω).

η social εκδοχή του Wayback Machine

Οι δυνατότητες που δίνονται μόλις κάνετε Archive μια σελίδα είναι να την εκδώσετε (να έχει ένα public url) ή να την κρατήσετε ιδιωτική. Για την κατηγοριοποίηση των σελίδων που αποθηκεύετε χρησιμοποιούνται: η αρχαία και δοκιμασμένη μέθοδος των folder αλλά και η νέα όχι ακόμα φτασμένη μέθοδος του tagging. Επίσης μπορείτε να αναζητήσετε ανάμεσα στις σελίδες που έχετε αποθηκεύσει ή ανάμεσα στις σελίδες όλων των χρηστών και τα καλά νέα είναι ότι ψάχνει και μέσα στις αποθηκευμένες σελίδες.

Όσο έχω δοκιμάσει τη σελίδα οφείλω να πω ότι λειτουργεί εξαιρετικά και είναι ό,τι πρέπει για διάσωση σελίδων η για να κρατάμε backup ή ό,τι άλλο μπορεί να προκύψει. Για παράδειγμα άλλαξα εμφάνιση στη σελίδα μου και θέλω κάπου να αποθηκεύσω το παλιό design. Ή είμαι Web Designer και θέλω να δείξω στο Portfolio μου δουλειές που έχουν αλλάξει. Ή φοβάμαι ότι μια σελίδα μπορεί να μην υπάρχει αύριο, οπότε θέλω να την κρατήσω ζωντανή.

Το Iterasi πάντως έχει ήδη και πολλούς Έλληνες χρήστες. Το ανακάλυψα μέσα από τα στατιστικά του Webz όπου κάποιος είχε αποθηκεύσει άρθρα μας.

Update: Ένα αρνητικό που μόλις εντόπισα είναι ότι το javascript δεν λειτουργεί στις αποθηκευμένες σελίδες.
